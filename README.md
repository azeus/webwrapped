# webwrapped
simple html file to show all websites visited in a day given your browser history


## How to use it:

1. Open Finder on your Mac
2. Press Cmd + Shift + G
3. Paste: ~/Library/Safari/
4. Copy the History.db file to your Desktop (or anywhere accessible)
5. Open this html file and upload your History.db file
6. Browse your history by clicking on any date!

Privacy : Everything remains in your local machine
